# nenrin-vis

**Visualizing Accumulation of Activities on Polar Coordinates.**
個人の活動の「厚み」を、年輪（Tree Rings）のような極座標系で可視化するビューワー。

## Concept

`nenrin-vis` は、単なる時系列プロットではなく、**活動の蓄積と密度**を極座標（Polar Coordinates）上に可視化するツールです。

樹木が環境や成長速度に応じて異なる幅の年輪を刻むように、このビューワーは「一定の時間経過」ではなく「活動量の累積」を半径方向の成長として表現します。
記録のない時間は圧縮され、充実した期間は分厚い層として刻まれるため、自身の歩んできた軌跡の「密度」を直感的な模様として捉えることができます。

## Core Logic

### 1. Radius represents "Volume" ($r \propto \sum Activity$)
本システムにおける中心からの距離（半径）は、単純な時間経過ではありません。**活動量の累積**を表します。

- **成長する年輪:** 記録やログが存在する期間のみ、外側へ向かって年輪が成長（層が形成）します。
- **可変する密度:** 活動が活発な時期は年輪の幅が「厚く」、停滞している時期や記録のない時期は「薄く（あるいは無く）」なります。

これにより、等間隔の時間軸では見えてこない「熱量のムラ」や「没頭した期間」が、視覚的な重みを持って表現されます。

### 2. Angle represents "Context" ($\theta \approx Domain$)
活動のジャンルやコンテキスト（例: 開発、研究、趣味、生活）によって角度を決定します。
ユーザー定義の「意味の近さ」に基づいて類似ドメインを隣接させることで、異なる活動間の相関や、興味の対象がどうシフトしていったかを可視化します。

### 3. Texture & Knots
データポイントは単なる点ではなく、質感を持ちます。

- **Layers (層):** 日々の記録は地層のように積み重なり、全体としてのテクスチャを形成します。
- **Knots (節):** 特筆すべき成果やイベントは、年輪上の「節（ふし）」として強調され、滑らかな層の中にアクセントを生み出します。

## Use Cases

- **Personal Chronicle:** 「何もしなかった期間」は省略され、「充実した期間」が強調されるバイオグラフィー。
- **Skill Growth:** 学習量（インプット/アウトプット）に応じて、特定領域が肥大化していくスキル可視化。
- **Project Density:** プロジェクトごとのリソース投下量やコミット密度の推移確認。
